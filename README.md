## Development Environment Setup Guide

Download the [NodeJS](https://nodejs.org/en) and install it.

# How to run code

## Clone the repository

```sh
git clone https://github.com/Bipon-Roy/react-task-QuadB
```

Now, open the <b>react-task-QuadB</b> folder in your <b>IDE (VS Code recommended)</b> and run the following commands on <b>terminal (Git Bash)</b>

<b>Using NPM</b>

```bash
  npm install
```

```bash
  npm run dev
```

You can run the local server on port [5173](http://localhost:5173/). Now, you should have accessed the following URLs on your local server.
